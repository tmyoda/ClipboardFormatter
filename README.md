# ClipboardFormatter
論文のpdfをgoogle翻訳かけるときに改行を取り除いてくれるスクリプト <br>
実行すると，クリップボードを監視してくれるので，いつものようにコピペするだけで勝手に改行が取り除かれます<br>
終了はCtrl+Cで行えます 

感度調整したいときは、ClipboardWatcherのpauseを調節すると良いです
# Required
- Python3
- pyperclip
- threading ( < Python 3.7)

# Usage
``` 
python main.py
```
